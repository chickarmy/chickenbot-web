## Version 1.0.0

Chickenbot release 1.0.0

<!-- truncate -->

### Description
La version 1.0.0 est la toute première version publique de ChickenBot !

ChickenBot est une application de Bureau Windows autonome qui permet de démarrer un joueur Minecraft automatisé : 
On parle ici de ’robot’, ou ’bot’ en anglais.

Le principal objectif de cette application est de permettre à un large public de créer simplement des robots Minecraft amusants.


Dans cette version initiale vous pouvez configurer le robot avec une ou plusieurs fonctions :
- fonction ’welcome’ (max:1) : indiquer le chat que prononce le robot à son apparition (ou réapparition),
- fonction ’reply’ (1 ou plusieurs) une ou plusieurs question/réponse: le robot détecte un chat particulier et répond avec un chat particulier,
- fonction ’follow-me’ (max:1) demander à votre robot de vous suivre, ou d’arrêter de vous suivre.

C’est une application Freemium. La version 1.0.0 est 100% gratuite.
Il est possible que dans de futures versions, certaines (nouvelles futures) fonctions soient payantes.

Voici enfin quelques précisions techniques et conditions d’utilisation.
Les interactions entre l’application et l’extérieur sont les suivantes:
- suivant la configuration du(des) robot(s), échanges avec le(s) serveur(s) Minecraft pour piloter leur connexion/actions,
- envoi de statistiques anonymisées d’utilisation de l’application vers une instance Cloud de Umami.

Utiliser l’application ChickenBot revient à accepter ces conditions d’utilisation et donc l’envoi de statistiques anonymisés.
En transparence, le guide installation montre comment activer le debug et tracer ces appels.
Les statistiques anonymisées servent à mieux connaître les usages de l’application, et à améliorer les futures versions.
